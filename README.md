# Unlimited Sampling Exploration

This repository contains an exploration of the paper "Unlimited Sampling Beyond Modulo" (Azar, Mulleti and Eldar, 2025)

It includes the main notebook `unlimited-sampling-b2r2.ipynb` and a cache of the experimental sweep results.

## AI Use Declaration
Generative AI tools were used to assist mainly in debugging, visualisations and table formatting, organizing code to reusable functions and for the caching of results. The AI-generated suggestions were reviewed, modified, and tested by the author to ensure correctness and relevance.
